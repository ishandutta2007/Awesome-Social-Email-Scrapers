## 🎯 Top Social Email Scrapers Ecosystem for Lead Generation

**Curated List of SaaS Products & Open-Source GitHub Projects**  
*Focused on LinkedIn, Twitter/X, Instagram & B2B Email Discovery Tools*  
**Last updated: March 2026**

This repository tracks notable **SaaS platforms** and **open-source projects** building **Social Email Scrapers**. These tools are essential for **lead generation**, finding professional emails from LinkedIn, Twitter/X, and other social platforms, enriching B2B data, and supporting outbound sales, recruitment, and growth marketing campaigns.

**Examples** include Apollo.io, Clay, Hunter.io, PhantomBuster, Prospeo, Skrapp.io, Snov.io, Socleads, Thunderbit, and ZoomInfo (the category leaders). Tools listed here emphasize **accuracy**, scale, enrichment, and compliance features.

**Open-source emphasis**: This section is heavily expanded with every major active project for self-hosting, ethical scraping, local execution, and full customization — ideal for developers and teams who want transparency, privacy, and control over data extraction.

## 💻 Open-Source GitHub Projects

### 🛠️ Dedicated Social Email & Lead Scrapers

- **[LinkedIn Email Extractors](https://github.com/search?q=linkedin+email+scraper)** 🖇️  
  Multiple community open-source tools for ethical LinkedIn profile email and contact extraction.

- **[PhantomBuster Open Alternatives](https://github.com/search?q=phantombuster+open+source)** 👻  
  Self-hosted automation scripts and actors replicating PhantomBuster-style social scraping.

- **[Hunter.io Style Finders](https://github.com/search?q=email+finder+open+source)** 🔍  
  Various open-source email discovery tools using pattern matching, GitHub, and public data sources.

- **[Playwright LinkedIn Scrapers](https://github.com/microsoft/playwright)** 🎭  
  Modern browser automation scripts for reliable social profile data extraction.

- **[Selenium Social Scrapers](https://github.com/SeleniumHQ/selenium)** 🧪  
  Classic open-source browser automation with many community repositories for LinkedIn/Twitter email finding.

- **[n8n Social Lead Workflows](https://github.com/n8n-io/n8n)** 🔄  
  Open-source workflow automation with LLM nodes for building ethical social email enrichment pipelines.

- **[Huginn Agents](https://github.com/huginn/huginn)** 🕵️  
  Personal automation agent for monitoring social profiles and extracting contact information.

- **[Social-Email-Finder](https://github.com/search?q=social-email-finder)** 📧  
  Collection of open-source tools combining public APIs, GitHub, and social signals for email discovery.

- **[Scrapy Social Spiders](https://github.com/scrapy/scrapy)** 🕷️  
  Powerful Python framework with many community spiders for ethical social media data collection.

### 🛠️ Additional Strong Open-Source Options

- **[Apify Actors](https://github.com/apify)** 🤖 — Open-source scraping actors for LinkedIn and social platforms.
- **[BeautifulSoup + Requests** 🍜 — Python scripts for lightweight public data extraction.
- **[LangGraph + Playwright Agents** 🧠 — Intelligent multi-step social lead enrichment workflows.
- **[Proxy Rotator + Scraper Stacks (Scrapoxy)](https://github.com/scrapoxy)** 🛡️ — For scalable, ethical scraping.
- Many **Ollama + RAG** pipelines 🧬 — For processing scraped social data and enriching emails locally.
